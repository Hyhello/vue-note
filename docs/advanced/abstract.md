# 抽象组件

#### 前言

抽象是从众多的事物中抽取出共同的、本质性的特征，而舍弃其非本质的特征的过程。具体地说，抽象就是人们在实践的基础上，对于丰富的感性材料通过去粗取精、去伪存真、由此及彼、由表及里的加工制作，形成概念、判断、推理等思维形式，以反映事物的本质和规律的方法。但是在程序中的表现形式是如何的呢？:clap:我们接下来往下看...

#### 抽象组件与普通组件的区别

- 抽象组件需要在声明组件是指定abstract。

- 抽象组件需包含一个元素
